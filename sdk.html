Quickly use WhatsApp API in your projects. All SDKs use automated code generation provided by openapi-generator. Unfortunately, openapi-generator has serious breaking changes in the generated code when upgrading versions. Therefore, we do not make backwards compatibility promises with regards to the generated SDKs. We hope to improve this process in the future. If this happens, we advise you to generate a new version of the SDK in your account.

Before you check out the SDKs, head over to the Whatsapp REST API documentation which includes code samples for common programming languages for each REST endpoint.
